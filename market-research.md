# Исследование рынка металлоконструкций в Барнауле (апрель 2026)

## Резюме

Рынок металлоконструкций для частного сектора в Барнауле активен и растёт. Производство готовых металлических изделий в Алтайском крае выросло на **+20,9%** по итогам 2025 года (данные Минпрома). Навесы для авто — самый высокочастотный запрос и основной продукт. Конкуренция умеренная: 5–8 основных игроков, большинство — федеральные агрегаторы с региональными страницами, а не локальные производства.

## 1. Анализ спроса по категориям (Яндекс Вордстат, январь 2026, РФ)

### Навесы для авто (ОСНОВНОЙ ПРОДУКТ)
- «навес для авто» — **6 156** запросов/мес
- «навес из металла» — **4 350** запросов/мес
- «навес из профтрубы» — **865** запросов/мес
- С локализацией «Барнаул» — около 150–300 запросов/мес (региональная доля ~3–5%)
- **Вердикт: высокий спрос, ярко выраженная сезонность (март–сентябрь)**

### Перголы
- «пергола купить» — среднечастотный (1000–3000/мес по РФ)
- В Барнауле предложений мало, рынок растущий
- Тренд 2025–2026: перголы «металл + дерево» набирают популярность для дач и загородных домов
- **Вердикт: низкая конкуренция, растущий тренд, хорошая маржинальность**

### Козырьки над крыльцом
- «козырёк над крыльцом» — **3 000–5 000** запросов/мес по РФ
- Стабильный спрос круглый год
- **Вердикт: хороший сопутствующий продукт, продаётся вместе с навесами**

### Лавочки / скамейки
- Спрос есть, но в основном от B2B (администрации, ТСЖ, благоустройство)
- Частные клиенты заказывают редко
- **Вердикт: дополнительный продукт для муниципальных заказов и комплексных проектов**

- Простой из поликарбоната: **от 3 237 руб/м²**

- Бюджетные (сталь + дерево): **2 000–10 000 руб**
- Парковые (сталь + дерево): **14 000–25 000 руб**

### Навесы для веранды (металл + дерево)
- Из металла: от **1 700 руб/м²**
- Из дерева: от **3 500 руб/м²**
- Комбинированный (металл + дерево): **слабо представлен, потенциал для премиум-цены**

## 5. Рекомендации по позиционированию

### Приоритет продуктов (по потенциалу)

1. **Навесы для авто** — основной хлеб, высокий спрос, быстрый оборот. Начинать с него
2. **Перголы металл + дерево** — низкая конкуренция, высокая маржа, тренд. Главная фишка для отстройки от конкурентов
3. **Навесы для террас/веранд (металл + дерево)** — расширение линейки, тот же клиент
4. **Козырьки над крыльцом** — хороший допродукт к навесам
5. **Лавочки и МАФ** — дополнительно, для комплексных проектов и B2B

### Ключевое конкурентное преимущество

Никто в Барнауле не делает акцент на **стиль «металл + дерево»** — это и перголы, и навесы для веранд, и лавочки. Большинство конкурентов предлагают стандартный металлический каркас + поликарбонат/профнастил. Красивые комбинированные конструкции — это свободная ниша.

### Каналы продаж (порядок приоритета)

1. **Авито** — главная площадка для частных заказов в Барнауле. Создать 5–10 карточек с фото готовых работ
2. **Яндекс Карты / 2ГИС** — зарегистрировать компанию, собирать отзывы
3. **Одностраничный сайт** — с калькулятором, портфолио, ценами
4. **ВКонтакте** — группа с фото процесса и готовых работ
5. **Яндекс Директ** — контекстная реклама по ключевым запросам (когда будет портфолио)

### Ключевые поисковые запросы для продвижения

**Высокий приоритет:**
- навес для авто Барнаул
- навес для машины Барнаул

- металлический навес Барнаул купить

**Средний приоритет (низкая конкуренция!):**
- пергола Барнаул
- пергола металл дерево Барнаул
- навес для веранды Барнаул

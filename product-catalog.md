## 1. НАВЕСЫ

### По форме крыши

| # | Форма | Схема | Описание | Сложность | Когда использовать |
|---|-------|-------|----------|-----------|-------------------|
| 1 | **Односкатный** | `╲________` | Одна плоскость под углом 10–30°. Самый простой и дешёвый | Низкая | Пристроенный к дому, для 1 авто, хозблок |
| 2 | **Двускатный** | `╱╲` | Два ската под углом 40–45°, как крыша дома | Средняя | Отдельно стоящий, для 2+ авто, большие площади |
| 3 | **Арочный** | `⌒` | Полукруглый. Красивый, снег не задерживается | Средняя | Авто, террасы, зоны отдыха, поликарбонат |
| 4 | **Прямой (плоский)** | `________` | Минимальный уклон. Только для южных регионов | Низкая | Декоративные, маленькие козырьки |
| 5 | **Вальмовый (4 ската)** | `⌂` (4 стороны) | Четыре ската — 2 трапеции + 2 треугольника | Высокая | Премиум, большие площади, беседки |

### По назначению и типовым размерам

| Назначение | Типовые размеры (Ш × Д) | Мин. высота столбов | Кол-во столбов | Кровля |
|------------|--------------------------|---------------------|----------------|--------|
| **Навес для 1 авто** | 3×5 / 3×6 / 3.5×6 м | 2,1–2,3 м | 4–6 | Профнастил / поликарбонат |
| **Навес для 2 авто** | 5.5×6 / 6×6 / 6×8 м | 2,3–2,5 м | 6–8 | Профнастил / поликарбонат |
| **Навес к дому (веранда)** | 3×4 / 3×6 / 4×6 м | 2,2–2,5 м | 2–4 (пристенный) | Поликарбонат / металлочерепица |
| **Навес для террасы** | 4×5 / 4×6 / 5×8 м | 2,3–2,7 м | 4–6 | Поликарбонат / мягкая кровля |
| **Навес для мангала/барбекю** | 3×3 / 3×4 м | 2,2–2,5 м | 4 | Профнастил / металлочерепица |
| **Навес хозяйственный** | 3×4 / 4×6 м | 2,0–2,3 м | 4–6 | Профнастил |

### По типу крепления

| Тип | Описание | Столбы |
|-----|----------|--------|
| **Отдельно стоящий** | На своих опорах, не связан со зданием | 4–8+ шт |
| **Пристенный (примыкающий)** | Одна сторона крепится к стене дома | 2–4 шт |
| **Консольный** | Без столбов, крепится кронштейнами к стене | 0 (только крепёж) |

## 2. КОЗЫРЬКИ

### По форме

| # | Форма | Описание | Для чего подходит |
|---|-------|----------|-------------------|
| 1 | **Односкатный** | Наклонная плоскость, 15–30°. Самый простой | Входы, крыльцо, бюджетные решения |
| 2 | **Двускатный** | Два ската «домиком». Надёжный водоотвод | Широкие входы, снежные регионы |
| 3 | **Арочный** | Полукруг. Элегантный, снег не копится | Крыльцо, балконы, витрины магазинов |
| 4 | **Купольный** | Полусфера. Декоративный, сложный | Классические фасады, премиум |
| 5 | **Маркизный** | Полуарка, выгнутая вперёд | Кафе, магазины, балконы |
| 7 | **Прямой (плоский)** | Горизонтальная плоскость с мин. уклоном | Минимализм, лофт |

## 5. ФОРМУЛА РАСЧЁТА СТОИМОСТИ

### Базовая формула

```
ЦЕНА ИЗДЕЛИЯ = Материалы + Работа + Доставка + Монтаж
```

### Расчёт площади

```
Площадь навеса = Длина × Ширина

Примеры:
  3 × 6 м = 18 м²
  6 × 6 м = 36 м²
  4 × 5 м = 20 м²
```

### Расход металла (профильная труба)

| Элемент | Сечение трубы | Расход на 18 м² (3×6) |
|---------|---------------|----------------------|
| Столбы (4–6 шт × 2,5 м) | 80×80×4 мм | 10–15 п.м. |
| Фермы / балки | 60×40×3 мм | 12–18 п.м. |
| Обрешётка | 40×20×2 мм | 20–30 п.м. |
| **Итого металла** | — | **200–250 кг** |

Для навеса большей площади: расход растёт примерно пропорционально, но на больших навесах нужны усиленные столбы (100×100 или 120×120 мм).

## 6. КАК ПОДАВАТЬ НА САЙТЕ

### Структура каталога для сайта

```
Навесы
├── Для автомобиля
│   ├── На 1 авто (3×5, 3×6, 3.5×6)
│   └── На 2 авто (5.5×6, 6×6, 6×8)
├── К дому (пристенные)
│   ├── Для веранды
│   └── Для террасы
├── Для мангала / барбекю
└── Хозяйственные

Перголы и беседки
├── Перголы (открытые, с навесом)
└── Беседки (открытые, полузакрытые, с мангалом)

Козырьки
├── Над крыльцом
├── Над входной дверью
└── Над балконом / окном
```

### Карточка товара — что показывать

Для каждого изделия на сайте:
1. Фото (фронт + сбоку)
2. Форма крыши (односкатный / арочный / двускатный)
3. Типовые размеры (3 варианта: S / M / L)
4. Материал каркаса + кровли
5. Цена «от» за типовой размер
6. Кнопка «Рассчитать мой размер»

### Калькулятор для сайта — входные параметры

```
1. Тип изделия: [навес / пергола / беседка / козырёк]
2. Назначение: [авто / веранда / терраса / крыльцо / ...]
3. Форма: [односкатный / двускатный / арочный]
4. Размер: Длина ___ м × Ширина ___ м
5. Материал кровли: [профнастил / поликарбонат / металлочерепица]
6. Дополнительно: [ковка / деревянная обрешётка / водосток]

→ Результат: ориентировочная стоимость
```
